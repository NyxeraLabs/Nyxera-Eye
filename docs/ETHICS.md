# Research Ethics Policy

Nyxera Eye research must prioritize safety, proportionality, and legal authorization.

## Core Principles

- Minimize potential harm to systems and people.
- Collect only data necessary for security analysis.
- Respect privacy and confidentiality obligations.
- Use the least-intrusive method that satisfies security objectives.

## Operational Guardrails

- Passive Mode is the default and preferred mode.
- Active probing is only permitted in Authorized Scope Mode.
- All non-passive activity must be attributable to an operator and logged.

## Data Handling

- Retain security-relevant metadata only.
- Protect stored artifacts and logs using access controls.
- Remove data on owner opt-out requests when legally applicable.
