# Responsible Security Disclosure

If Nyxera Eye identifies exposed infrastructure that may create risk, disclose findings responsibly.

## Disclosure Flow

1. Validate the finding with non-intrusive methods.
2. Identify the most appropriate owner contact channel.
3. Provide concise technical evidence and reproduction context.
4. Avoid publication before reasonable remediation time.

## Safe Reporting Rules

- Never include credentials or sensitive data in reports.
- Do not exploit vulnerabilities to prove impact.
- Share only the minimum evidence required for remediation.

## Contact

For disclosure coordination, use: `security@nyxera.cloud`.
